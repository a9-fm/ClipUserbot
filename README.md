# ClipUserbot

---


## Для Termux (Android)

Установка
```
 apt-get update -y ; apt-get install python -y ; apt-get install git -y ; git clone https://github.com/A9FM/ClipUserbot ; cd ClipUserbot ; python3 bot.py
```

При запуске
```
cd ClipUserbot && termux-wake-lock && python bot.py
```
